# n8n AI Automation

## What this project does
This repository contains simple AI automation workflows built using n8n.
The example workflow demonstrates how to call an AI model and process text automatically.

## Workflow included
- Text summarization using AI
- Environment-variable based API key handling

## Requirements
- n8n
- Docker (optional)
- AI API key (Gemini / OpenAI)

## How to run
1. Clone the repository
2. Copy `.env.example` to `.env`
3. Add your API key
4. Start n8n
5. Import `workflow.json`

## Notes
This project is for learning and demonstration purposes.
